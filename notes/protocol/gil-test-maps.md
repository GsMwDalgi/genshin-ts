# GIL Test Map Registry

GIL은 에디터가 디스크에 저장하는 맵 파일 포맷. 인젝션/바이너리 비교 분석의 기준 파일로 사용.

|------------|--------------------------------|-----------------------------------|------------------------|
| 1073741956 | Sample-empty                   | 초기 상태의 완전 새로운 빈 맵      | 기준선 (baseline)       |
| 1073741957 | Sample-signal_args_test        | signal_args_test.ts 기반 신호 노드를 에디터에서 기본 생성한 맵. 빈 노드 그래프 포함 | 신호 등록 확인용 |
| 1073741958 | Sample-signal_args_test_injected | Sample-signal_args_test를 복사한 맵. 인젝션 대상 | **인젝션 테스트 대상** |
| 1073741960 | temp                           | 이펙트 컴포넌트 분석용. 오브젝트 1077936130에 이펙트 4개(10004139~10004142) 등록 | **GIL 구조 분석 대상** |

## Binary Comparison Methodology

1. **에디터 기준**: 1073741957의 GIL을 protobuf 디코딩하여 에디터가 생성한 노드 구조 확인
2. **인젝션 결과**: 1073741958에 인젝션 후 GIL을 디코딩하여 비교
3. **빈 기준선**: 1073741956으로 GIL 파일 기본 구조 파악
